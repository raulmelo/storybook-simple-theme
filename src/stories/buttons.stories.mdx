# Transparent

Para adicionar botões sem cor de fundo.
* .bt-outline

```js
<button class="bt bt-sm bg-primary bt-outline">Botão</button>
```

export const myStyles = [
`.group-buttons button {
    margin-left: 20px
  }
  .group-buttons {
    display: flex;
    align-items: center;
    justify-content: center;
    margin-top: 50px;
  }
`]

# Botão em loading

Para simular ação de loading no botão adiciona a seguinte classe css
* .loading

```js
<button class="bt bt-sm bg-primary loading">Botão</button>
```

# Botão com ícone

Para adicionar botão com ícone botão adiciona a seguinte classe css
* .bt-icon

```js
  <button class="bt bt-md bt-icon bg-primary bt-radius">
    Botão
    <i class="icon-megafone"></i>
  </button>
```

# Botão com ícone circle
O botão circle é um tipo de botão arredondado e com um ícone para demonstrar o componente
deve adicionar algumas tags por conta de acessibilidade como _alt_, _title_ e _role_

* .bt-circle

```js
  <button class="bt bt-lg bt-circle bg-primary">
    <i class="icon-megafone"></i>
  </button>
```

# Botão Label
O botão label foi criado com intuito de representar tags ou elementos dentro de cards que executam determinadas ações, para ser usado no botão tem que adicionar a seguinte classe no *button*.;

* .bt-label

```js
  <button class="bt bt-lg bt-label bt-radius bg-secondary bt-outline">ver checklist</button>
```
