# Simples

O user picture é um componente de avatar de acordo com a imagem passada se o não tiver imagem ele traz as iniciais do nome do usuário passado

```html
  <user-picture size="sm" name="Raul Melo" image="https:minhasimagem.url.com" profile="Asven"></user-picture>
```

export const Default = (args) => ({
  props: args,
  template: `
    <user-picture [size]="size" [name]="name" [image]="image"></user-picture>
  `
})

__
```js
## Props
@Input() name: string;
@Input() image: string;
@Input() profile: string = 'asven';
@Input() size: 'sm' | 'md' | 'lg' = 'sm';
```

| PROP | Type | Default | Descr |
| -    |  -   |    -    | -     |
| name | string | null | essa propriedade serve para que se a imagem não for renderizada ele mostra o nome abreviado
| image | string | null | link da imagem que sera mostrada como avatar |
| profile | string | asven | por padrão vem com as cores padrões do asven, porem pode passar outras propriedades e custumizar da sua forma
| size  | 'sm', 'md', 'lg' | sm | Tamanhos já definidos para o componente
